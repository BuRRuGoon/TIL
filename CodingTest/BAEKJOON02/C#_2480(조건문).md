# 예제 정수 입력 문제

## 내가 푼 방식
``` cs
int[] x = Array.ConvertAll(Console.ReadLine().Split(), int.Parse);

int count = x.Distinct().ToArray().Count();

if (count == 1)
{
    Console.Write(10000 + (x[0] * 1000));
}
else if (count == 2)
{
    if (x[0] == x[1] || x[0] == x[2])
    {
        Console.Write(1000 + (x[0] * 100));
    }
    else 
    {
        Console.Write(1000 + (x[1] * 100));
    }
}
else
{
    Console.Write(x.Max() * 100);
}
```
