# 예제 출력 문제

## 내가 푼 방식
``` cs
Console.WriteLine("\\    /\\");
Console.WriteLine(" )  ( ')");
Console.WriteLine("(  /  )");
Console.WriteLine(" \\(__)|");
```

## 다른사람이 푼 방식
``` cs
Console.Write(@"\    /\
 )  ( ')
(  /  )
 \(__)|");
```

WriteLine 대신 Write를 이용해서 한번에 출력하는 방식
메모리나 시간상에는 큰 차이가 없음
C#은 다른 언어에 비해서 메모리 사용량이 높음
